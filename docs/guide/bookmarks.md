# Bookmarks

Save frequently-used subscriptions, publish templates, and message bookmarks for quick access.

## Saved Subscriptions

In the Pub/Sub panel's Subscribe tab, click the star icon next to any active subscription to save it. Enter a name and confirm.

Saved subscriptions appear in a "Saved" bar above the subscribe input. Click one to instantly subscribe to that subject.

## Publish Templates

In the Publish tab, fill in the subject, headers, and payload, then click **Save Template**. Enter a name for the template.

Templates appear at the top of the Publish tab. Click a template name to load its subject, headers, and payload. Templates persist across sessions.

## Message Bookmarks

In the Message Browser, select a message and click **Bookmark** in the detail pane. Enter a name to save a reference to that stream + sequence number.

## Storage

All bookmarks are stored in VS Code's globalState — they persist across sessions but are local to your VS Code installation. Bookmarks are not synced via Settings Sync.
