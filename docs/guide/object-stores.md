# Object Stores

Browse NATS Object Store contents.

## Browsing

The Object Stores view lists all stores with object counts and sizes. Expand a store to see its objects.

## Object Viewer

Click an object to see its metadata:
- Name, size, chunk count
- Last modified timestamp
- Content digest

## Management

- **Delete** — remove an object (with confirmation)
- **Refresh** — reload the object list
