# Settings

## `leafnode.autoRefreshInterval`

- **Type:** `number`
- **Default:** `30`
- **Description:** Auto-refresh interval for tree views in seconds. Set to `0` to disable auto-refresh.

Tree views (Streams, KV Stores, Object Stores, Subjects) refresh on this interval when a connection is active.

## Keyboard Shortcuts

Leafnode registers the following default keyboard shortcuts. These can be customized in **File > Preferences > Keyboard Shortcuts**.

| Shortcut | Command |
|----------|---------|
| `Ctrl+Shift+Alt+N` (`Cmd+Shift+Alt+N` on macOS) | Add Connection |
| `Ctrl+Shift+Alt+M` (`Cmd+Shift+Alt+M` on macOS) | Open Pub/Sub Panel |
| `Ctrl+Shift+Alt+J` (`Cmd+Shift+Alt+J` on macOS) | Open Server Monitor |

See the full [command reference](/reference/commands) for all available commands.
